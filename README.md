# Calculator

A simple, responsive calculator built with vanilla HTML, CSS, and JavaScript. It supports basic arithmetic operations and keyboard input for a smooth user experience.

## Features

- Addition, subtraction, multiplication, and division
- Clear and delete functions
- Keyboard support for numbers, operators, and commands
- Responsive design with dynamic display resizing

## Getting Started

No build step is required. Open `index.html` in any modern web browser to start using the calculator.

## Live Demo

- GitHub Pages: https://paulcaliguid.github.io/calculator/

## Project Structure

- `index.html` – markup for the calculator interface
- `style.css` – styles and layout
- `script.js` – application logic and keyboard handling
